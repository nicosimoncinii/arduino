## 1. 💡 Led 

Far lampeggiare un LED a intervalli regolari utilizzando Arduino.

### ⚙️ Funzionamento

Il LED si accende e si spegne continuamente grazie alla funzione `delay()`.

---

## 2. 🚦 Semaforo

Simulazione di un semaforo con tre LED (rosso, giallo, verde) che si alternano automaticamente.

### ⚙️ Funzionamento

Il semaforo segue un ciclo:

* Verde → Giallo → Rosso

---

## 3. 🔘 Semaforo con Pulsante

Semaforo che cambia stato alla pressione di un pulsante.

### ⚙️ Funzionamento

Il ciclo del semaforo viene controllato manualmente:

* Alla pressione del pulsante → cambia stato

---

## 4. 🚧 Incrocio

Simulazione di un incrocio stradale con 4 semafori sincronizzati.

### ⚙️ Funzionamento

* I semafori opposti funzionano insieme
* Le due direzioni si alternano

### 🔄 Ciclo

1. Direzione A → Verde | Direzione B → Rosso
2. Direzione A → Giallo
3. Direzione B → Verde | Direzione A → Rosso
4. Direzione B → Giallo
